# PerformanceBooster
PerformanceBooster优化mod
  Performance Booster 包含以下优化：                                                                                                                             
                                                                                                                                                                 
  - 实体渲染优化 — 超过48格的实体自动隐藏，减少渲染负载                                                                                                          
  - 粒子优化 — 限制每tick粒子数量（500上限），跳过烟雾/粉尘等重型粒子                                                                                          
  - Tick优化 — 方块实体更新降频（每2tick更新一次），降低CPU开销                                                                                                  
  - 区块优化 — 减少区块加载/卸载造成的卡顿                                                                                                                       
  - 渲染优化 — 距离裁剪，减少过绘
